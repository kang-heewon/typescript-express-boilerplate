# Typescript expres boilerplate

- use typescript
- [Test Driven Development](https://github.com/dwyl/learn-tdd)
- git semantic commit message 사용
  > 윈도우: windows key + ;
  > mac : ctrl + command + space

| 언제?                              | 커밋메시지                              |
| ---------------------------------- | --------------------------------------- |
| 기능추가                           | feat: ⚡️ 기능추가                      |
| 버그픽스                           | fix: 🔥 버그픽스                        |
| 리팩토링                           | refactor: 🛠 리팩토링                    |
| 프로덕션과 상관없는 패키징 등 코드 | chore: 📦 패키지추가, 배포스크립트 등등 |
| 리드미 추가                        | docs: 📚 리드미 수정                    |
